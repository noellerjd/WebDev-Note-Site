Nesting and Indentation

You can nest <ul> and <ol> within each other to make a more detailed list which could look like: 


              <ul>
                  <li>Unordered list item 1</li>
                      <ul>
                          <li>Unordered List Item 1.1</li>
                          <li>Unordered List Item 1.2</li>
                      </ul>
                  <li>Unordered list item 2</li>
                  <li>Unordered list item 3</li>
                      <ul>
                          <li>Unordered List Item 3.1</li>
                          <li>Unordered List Item 3.2</li>
                      </ul>
              </ul>
          
              <ol>
                  <li>Ordered list item 1</li>
                      <ol>
                          <li>Ordered List Item 1.1</li>
                          <li>Ordered List Item 1.2</li>
                      </ol>
                  <li>Ordered list item 2</li>
                  <li>Ordered list item 3</li>
                      <ol>
                          <li>Ordered List Item 3.1</li>
                          <li>Ordered List Item 3.2</li>
                      </ol>
                </ol>
